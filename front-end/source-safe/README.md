Source-Safe - Client Side
===

## How to run

First, confirm that you have Node and npm on your system:
```shell
node --version
```
```shell
npm --version
```

Then install the dependencies:
```shell
npm i
```
Now you're ready to run:

```shell
npm run dev
```
